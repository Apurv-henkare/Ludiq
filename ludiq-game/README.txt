LUDIQ Game - love.js Build
===========================

To make the game playable on this site:

1. Build the LUDIQ game with love.js:
   - Use https://github.com/Davidobot/love.js
   - Run: npx love.js ludiq_game.zip ludiq-build -c
   
2. Copy the generated game.data file here:
   - ludiq-game/game.data

The love.js, love.wasm, game.js, and index.html 
are already set up and ready to go.
